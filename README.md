Care Connect - README

Project Description:
Care Connect is a platform designed to connect users with NGOs (Non-Governmental Organizations) to facilitate donations, volunteering, and participation in charity events. The application allows users to donate goods (clothes, food) or money, view the requirements of NGOs, volunteer for charity drives, and rate/review the NGOs they interact with. NGOs can post their needs for donations, organize charity drives, and maintain profiles. The Admin can verify NGO registrations, approve drives, and monitor platform activities.

App Entities:
- Users: Can donate (clothes, money, or food), view NGOs' descriptions, ratings, and needs, volunteer for drives, and participate in donation campaigns or events.
- NGOs: Can post donation requirements, organize drives, coordinate with admins, and maintain profiles.
- Admin: Manages and verifies NGO registrations, approves drives, monitors platform activity, and resolves issues between users and NGOs.

Features:
1. **User Features**:
   - Registration/Login
   - Dashboard to view active drives, nearby NGO requirements, and personal donation history
   - Donation options (online payment for money, location pickup for goods)
   - Volunteer for NGO-organized drives
   - Rate and review NGOs after participation

2. **NGO Features**:
   - Registration/Login (verified by Admin)
   - Post donation requirements (type, quantity, area of need)
   - Organize drives (set location, time, agenda)
   - Track participants and coordinate with Admin for event approval
   - Maintain profile with description, contact details, and ratings/reviews

3. **Admin Features**:
   - Dashboard to monitor donations, user activity, and NGO requests
   - Approve or decline NGO and drive registrations
   - Communication channel with NGOs for event coordination
   - Generate platform analytics (e.g., most active NGOs, popular donation types)

Technologies Used:
- Flutter (Mobile App Development)
- Firebase (Authentication, Firestore Database, Push Notifications)
- GitHub (Version Control)
